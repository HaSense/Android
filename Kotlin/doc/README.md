# 마크다운(MarkDown) 사용방법

## 1. 제목
H1 ~ H6까지 작성하는 방법  

```shell
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

# H1
## H2
### H3
#### H4
##### H5
###### H6


## 2. 강조

강조할 텍스트를 다음과 같이 작성합니다.   
이탤릭체, 볼드체, 볼드체와 이탤릭체 결합

```shell
*italic*
**bold**
***bold and italic***
```

*italic*
**bold**
***bold and italic***
